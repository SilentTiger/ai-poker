## 产品是什么

一个会说人话的 AI 德扑教练。随时能问、能练、能对战，帮你从靠感觉打牌变成真正懂在做什么。

## 核心功能

### 模块零：专业术语学习及基本技术指导（区别其他产品）

- 是什么：根据用户水平自动调整 AI 说话方式，新手说人话，高手用术语，用户不需要做任何设置。
- 解决什么：新手被术语吓跑、高手觉得解释啰嗦，两个问题同时解决。
- 怎么判断水平：首次进入做五道测试题，也可手动选。答对率持续走高自动升档，语言跟着变。
- 术语处理：第一次出现旁边有小问号，点开是一句话解释，第二次出现后不再提示。

### 模块一：情景练习 + AI 即时解析

- 是什么：平台预设覆盖各难度、各街道的真实牌局情景，用户做决策，AI 立刻用自然语言反馈对错和原因。
- 解决什么：打完一手不知道错哪，只能靠感觉复盘。
- 核心体验：看到牌面 → 做决策 → AI 立刻反馈 → 看懂原因 → 下一题

#### 难度分级

| 级别 | 适合用户 | 情景特征 |
| --- | --- | --- |
| 入门 | 新手 | 翻牌前决策为主，附术语解释 |
| 基础 | 新手进阶 | 翻牌后简单场景，一次只变一个条件 |
| 进阶 | 中级 | 跨多轮综合判断，开始讲期望值的概念 |
| 高阶 | 高手 | 复杂多人局，分析和理论最优策略的偏差 |

### 模块二：AI 对手实战演练

- 是什么：用户与 AI 模拟玩家打多手实战。AI 对手有不同风格，模拟真实牌桌压力，关键手牌自动标记，局后进入复盘。
- 解决什么：有理论但缺实战演练，真人牌桌成本高、手数少。
- 核心体验：选对手风格 → 打一局（20–50轮）→ 关键手牌被标记 → 局后看 coaching
- 局后复盘：问题手牌按损失排序，每手可点进去看 AI 分析，最后总结本局反复出现的问题类型。

#### AI 对手风格（MVP）

| 风格 | 特征描述 | 训练目标 |
| --- | --- | --- |
| 松凶型 | 什么牌都跟，动不动就加注 | 练习在对方一直施压时，怎么判断该跟还是该放 |
| 紧弱型 | 很少参与，拿到好牌也不敢下注 | 练习在对方明显很弱时，怎么把钱赢出来 |
| 平衡型 | 打法接近理论最优，很难被读出规律 | 给打得不错的玩家测试自己的水平 |
| 鱼型 | 随机跟注，不管赔不赔得来 | 练习针对乱打玩家时怎么调整策略 |

### 模块三：赛后复盘 + 个性化训练推送

- 是什么：打完一局，AI告诉你这局哪里出了问题。打开产品，AI告诉你今天该练什么。
- 解决什么：前三个模块产生的数据如果不被利用，每次练习结束就归零。用户不知道自己反复犯什么错，不知道下次该练什么，进步靠运气。这个模块把数据变成理解，把理解变成行动。
- 运作：复盘。局结束 → 列出问题手牌（按损失排序）→ 点进去看AI解析 → 底部一句话总结这局的规律性问题。
- 运作：推送。下次打开产品 → 「你过去20局有6次翻牌后该下注没下，今天练这个」→ 3–5道对应的题。
- 核心原则：主动说，不等用户来问。
- 核心原则：说原因，不只说结论。
- 核心原则：数据跨局积累，不单局遗忘。
- 核心原则：复盘要轻，找出2–3手最值得看的。
- 核心原则：推送要精，3–5道够了。

## 产品边界

不做什么！！

| 不做 | 原因 |
| --- | --- |
| 打牌时实时给提示 | 所有正规平台都不允许，会被封号 |
| 自动替用户打牌的机器人 | 同上，而且和"帮用户变强"的目标完全反着来 |
| 只输出数字的求解工具 | 别人已经做了，我们的重点是把结果解释清楚，不是自己去算 |
